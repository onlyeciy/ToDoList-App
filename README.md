# To-Do List - App (JavaScript personal project)

## Overview
This project is a simple To-Do List system built using plain JavaScript (No UI).  
It focuses on core programming concepts such as:
- Arrays
- Objects
- Functions
- State management

The goal is to simulate how a basic task manager works internally.

---

## Features

### 1. Add Task
- Add a new task into the to-do list
- Each task has:
  - id (auto-increment)
  - task (string)
  - status ("incomplete" by default)

---

### 2. Remove Task
- Remove a task by its id

---

### 3. Mark Task as Completed
- Change task status to "completed"
- Move task into a separate completedTask array
